# toBaseFromImage

이미지 파일을 선택해서 HTML에서 바로 사용할 수 있는 Base64 Data URL로 변환하는 간단한 정적 페이지다.

## 파일 구성

- `image-to-base64.htm`
  - 이미지 선택
  - Base64 변환
  - 클립보드 복사
- `test.html`
  - 기존 보관 파일

## 사용 방법

1. `image-to-base64.htm`을 브라우저에서 연다.
2. 이미지 파일을 선택한다.
3. `Base64 변환` 버튼을 눌러 Base64 문자열을 생성한다.
4. 필요하면 `클립보드 복사` 버튼으로 복사한다.

## 비고

- 브라우저 보안 정책상 문자열 형태의 로컬 파일 경로를 직접 읽을 수는 없다.
- 따라서 `input type="file"` 기반으로 이미지를 선택해 변환하는 방식이 표준적인 방법이다.
- `test.html`은 Base64 변환 결과 확인용 예시 파일이며, 내장 이미지 데이터가 포함되어 파일 크기가 큰 편이다.
